# MLB HOF
This is a machine learning project to determine who will make the Baseball Hall of Fame.
The idea is to use data of previous inductees and players on the ballot to see if the information can be extrapolated to other players.

# Data
This project utilizes the [Lahman dataset](https://www.seanlahman.com/baseball-archive/statistics/).
A huge shoutout to Sean Lahman for putting this free and open source collection of data together.
This project would not have been possible without the awesome community of baseball lovers!

The Lahman data set contains a csv file titled `HallOfFame.csv`.
At the time of writing, the most recent data available from the Lahman dataset includes statistics from the 2021 season.
The `HallOfFame.csv` only contains data up to 2018, making it slightly out of date.
